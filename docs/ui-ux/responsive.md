## Touch Considerations

### Touch Targets
```css
/* Minimum touch target size */
.touch-target {
  min-width: 44px;
  min-height: 44px;
  padding: 12px;
}
```

### Touch-Specific Interactions
- Use `tap` instead of `hover` for mobile
- Implement swipe gestures for navigation where appropriate
- Provide visual feedback on touch (highlight state)
- Avoid hover-only interactions

### Bottom Action Bar (Mobile)
```css
/* Fixed bottom action bar for mobile */
.bottom-action-bar {
  position: fixed;
  bottom: 0;
  left: 0;
  right: 0;
  height: 64px;
  background-color: white;
  border-top: 1px solid var(--gray-200);
  display: flex;
  align-items: center;
  justify-content: space-around;
  padding: 0 16px;
  padding-bottom: env(safe-area-inset-bottom);
  z-index: var(--z-20);
}

/* Ensure content doesn't hide behind bar */
.page-content-with-bottom-bar {
  padding-bottom: calc(64px + env(safe-area-inset-bottom));
}
```

## Grid Systems

### Patient Card Grid
```css
/* Responsive patient card grid */
.patient-grid {
  display: grid;
  gap: 16px;
  grid-template-columns: 1fr;
}

@media (min-width: 768px) {
  .patient-grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (min-width: 1024px) {
  .patient-grid {
    grid-template-columns: repeat(3, 1fr);
    gap: 24px;
  }
}

@media (min-width: 1280px) {
  .patient-grid {
    grid-template-columns: repeat(4, 1fr);
  }
}
```

### Form Grid
```css
/* Responsive form grid */
.form-grid {
  display: grid;

@media (min-width: 768px) {
  .form-grid {
    grid-template-columns: repeat(2, 1fr);
  }

.form-grid .full-width {
```

## Responsive Testing Checklist

### Mobile (iPhone SE - 375px)
- [ ] All content visible without horizontal scroll
- [ ] Touch targets are 44px minimum
- [ ] Forms are usable with on-screen keyboard
- [ ] Navigation accessible via hamburger menu
- [ ] Text is readable without zooming

### Tablet (iPad - 768px)
- [ ] Two-column layouts where appropriate
- [ ] Sidebar collapsible and functional
- [ ] Forms utilize horizontal space
- [ ] Transcription panel slides correctly

### Desktop (MacBook - 1440px)
- [ ] Full sidebar visible
- [ ] Multi-column layouts displayed
- [ ] Side-by-side panels work correctly
- [ ] Hover states function properly

### Cross-Device
- [ ] Orientation changes handled (portrait/landscape)
- [ ] Content reflows correctly at all breakpoints
- [ ] Images scale appropriately
- [ ] Animations respect reduced-motion
