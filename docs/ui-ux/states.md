```typescript
    <p className="text-gray-500 max-w-md mb-6">
      {description}
    </p>
    {action && (
      <Button onClick={action.onClick}>
        {action.label}
      </Button>
    )}
  </div>
);
```

## Best Practices

### Error Messages

1. **Be specific** - Tell users exactly what went wrong
2. **Be helpful** - Suggest how to fix the problem
3. **Be human** - Use friendly, non-technical language
4. **Be brief** - Keep messages short and scannable

**Good:**
> "El email 'test@' no es vÃ¡lido. Ejemplo: usuario@correo.com"

**Bad:**
> "Error 422: Validation failed for field 'email'"

### Empty States

1. **Explain the context** - Why is this empty?
2. **Guide to action** - What can they do next?
3. **Be encouraging** - Frame positively
4. **Show visual** - Icon or illustration helps

### Loading States

1. **Show progress** - If possible, show percentage
2. **Maintain layout** - Use skeletons matching final content
3. **Keep interactive** - Allow cancel when possible
4. **Set expectations** - "This may take a moment"
