## Instrucciones para capturar

1. Iniciar la aplicación en desarrollo: `pnpm dev`
2. Acceder a http://localhost:5173
3. Iniciar sesión con credenciales de prueba
4. Capturar cada pantalla con dimensiones consistentes (1280x800 recomendado)
5. Nombrar archivos siguiendo la convención numérica

## Herramientas recomendadas

- **macOS**: Cmd + Shift + 4 (área) o Cmd + Shift + 5 (opciones)
- **Windows**: Win + Shift + S (Snip & Sketch)
- **Chrome DevTools**: Device toolbar para dimensiones consistentes
- **Firefox**: Screenshot tool integrado

## Formato

- Formato: PNG
- Dimensiones: 1280x800 píxeles (recomendado)
- Sin información personal visible
- Datos de prueba únicamente
